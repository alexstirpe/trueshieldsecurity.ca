# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-28
### Added
- Created `index.html` with a clean, modern layout for TrueShield Security, including:
  - Header with logo placeholder and navigation links (Home, About, Services, Contact) with sticky top behavior.
  - Hero section with a bold headline, subtext, and a CTA button with `id="cta-hero`.
  - Services preview section featuring three service cards (Mobile Patrol, Event Security, Concierge & Access Control) with descriptions and links to `services.html`.
  - About preview section with an image placeholder, summary text, and a button linking to `about.html`.
  - Contact CTA section with a bold callout message and a button linking to `contact.html`.
  - Footer with copyright notice, navigation links, and a placeholder for social icons.
  - Linked `assets/css/style.css` and `assets/js/main.js` for styling and interactivity.

- Created `about.html`, `services.html`, and `contact.html` as placeholders for respective pages.

- Established the `assets/` directory structure:
  - `assets/css/style.css`: Placeholder for the main stylesheet.
  - `assets/js/main.js`: Placeholder for the main JavaScript file.
  - `assets/img/`: Directory for storing image assets (currently empty).

### Notes
- The project follows a mobile-first, responsive design approach.
- Semantic HTML5 elements are used for better accessibility and SEO.
- Placeholder content and classes are included for future styling and functionality.
