# True Shield Security Services — Official Website

**Repository:** `trueshieldsecurity.ca`
**Project Type:** Client website (static front-end, extensible architecture)

---

## Overview

This repository contains the official website for **True Shield Security Services**, a locally owned and operated manned security services provider based in Ontario, Canada.

The website is intentionally built with **clean, standards-compliant HTML, CSS, and JavaScript** to ensure:

* Long-term maintainability
* Strong performance and accessibility
* Ease of future expansion (analytics, client portals, secure forms)

The project also serves as a **professional portfolio reference**, demonstrating real-world client collaboration, requirements gathering, and brand-aligned execution.

---

## Project Scope

### Current Features

* Static informational website
* Brand-aligned visual identity
* Responsive layout for desktop and mobile
* Semantic HTML structure
* Modular CSS organization
* Lightweight JavaScript for UI behavior

### Planned / Extensible Features

* JavaScript-based analytics integration
* Secure contact and inquiry forms
* Authentication-ready structure for client or staff portals
* Backend/API integration readiness

> The repository structure is intentionally designed to support incremental upgrades without requiring a full rewrite.

---

## Technology Stack

* **HTML5** — semantic, accessible markup
* **CSS3** — responsive layout and styling
* **Vanilla JavaScript (ES6+)** — client-side behavior

No frameworks are currently used to preserve simplicity, portability, and long-term client ownership.
