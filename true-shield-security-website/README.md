# True Shield Security Website

## Overview
True Shield Security is a professional security company dedicated to providing reliable and trustworthy security services. This website serves as a platform to showcase our services, mission, and values, allowing potential clients to learn more about what we offer and how to contact us.

## Project Structure
The project is organized as follows:

```
true-shield-security-website
├── index.html          # Home page with hero section and service overview
├── about.html          # Company mission, vision, and core values
├── services.html       # Detailed descriptions of services offered
├── contact.html        # Contact form and company information
├── css
│   └── styles.css      # Global stylesheet for styling the website
├── assets
│   ├── images          # Directory for stock images
│   └── fonts           # Directory for custom fonts
├── .gitignore          # Files and directories to ignore in version control
└── README.md           # Project documentation
```

## Setup Instructions
1. **Clone the Repository**
   - Use the command: `git clone <repository-url>`

2. **Navigate to the Project Directory**
   - `cd true-shield-security-website`

3. **Open the Project**
   - Open the project in your preferred code editor.

4. **View the Website**
   - Open `index.html` in a web browser to view the home page.
